Acerca de Nexus

✦

Nuestra misión

Nexus es una plataforma de blog dinámica construida sobre la API de DummyJSON como parte del curso de Sistemas y Tecnologías Web de la Universidad del Valle de Guatemala.

⬡

Tecnologías

HTML5 semántico, CSS con variables y diseño responsive, JavaScript modular con ES Modules, DummyJSON API para publicaciones y usuarios.

◈

Funcionalidades

Listado con paginación, búsqueda y filtros avanzados, detalle de publicación, formulario con validación, eliminación visual y experiencia de usuario fluida.
